# Data visualization

### Objective
* Display of networked nodes with qualitative links

### Inputs
* Nodes in `CSV` file
* Links in `CSV` file

### Tools
* `networkD3` library

### Output
* A vector graph showing the nodes and links with the numerical details on mouseover event

| File name | File type |
| ------------ | ------------- |
| MaskedNet1.html | HTML |
| Output_PNG.png | PNG |
| Output_SVG.svg | SVG |
